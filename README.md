# ShortURL in PHP 
ระบบย่อลิงค์ด้วย PHP

## Function :

 - ย่อ URL
 - แก้ไข URL
 - สร้าง qr code ของ URL แบบสั้น

## สิ่งที่ต้องเตรียม

 - PHP
 - My SQL
 - XAMPP

## วิธีติดตั้ง

 - ติดตั้ง XAMPP 
 - สร้าง Database โดยอัพโหลดไฟล์ชื่อ url.sql
 - ดาวน์โหลดไฟล์โปรเจคไว้ที่ โฟลเดอร์ \htdocs
 - รัน Apache และ MySql 
 - เปิด localhost/"folder porject" ผ่าน browser
